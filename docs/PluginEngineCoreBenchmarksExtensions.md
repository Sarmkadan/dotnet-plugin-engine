# PluginEngineCoreBenchmarksExtensions

Provides a set of benchmarking utilities for measuring the performance and reliability of `PluginEngineCore` operations. These extension methods encapsulate common benchmarking scenarios, including plugin loading, unloading, lifecycle management, and status verification, with built-in warm-up and cleanup phases to ensure consistent and reproducible results.
